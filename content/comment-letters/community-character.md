## Talking Points

- The project's scale and industrial use are inconsistent with the
  **[General Plan / Specific Plan]**'s vision for this corridor, which
  designates the area for **[intended use]**.
- Nighttime operations, including truck loading and exterior lighting, would
  affect neighboring residences' quiet enjoyment of their homes.
- Approving this project could set a precedent for further industrial
  encroachment into adjacent residential areas.
- Request findings that clearly document consistency (or inconsistency) with
  the General Plan and zoning code before any approval.
- Request conditions limiting operating hours, requiring shielded/downward
  lighting, and screening/buffering (landscaping, setbacks) from residential
  properties.
